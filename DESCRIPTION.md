# Description technique

## Objectif

Créer un portfolio professionnel moderne permettant de présenter :

- mon parcours
- mes compétences
- mes projets
- mes moyens de contact

---

# Architecture

Le projet repose sur :

HTML5
CSS3
JavaScript Vanilla

Aucun framework JavaScript n'est utilisé.

Le design repose principalement sur Tailwind CSS.

---

# Responsive

Le site est optimisé pour :

- Mobile
- Tablette
- Desktop
- Écrans larges

---

# Formulaire de contact

Le formulaire possède :

- validation côté client
- EmailJS
- WhatsApp
- vérification des emails
- validation du téléphone

---

# Accessibilité

Le projet respecte notamment :

- HTML sémantique
- alt des images
- labels associés
- navigation clavier
- lazy loading

---

# Performance

Optimisations :

- animations GPU
- lazy loading
- images optimisées
- CSS organisé
