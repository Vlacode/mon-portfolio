# 🌐 Portfolio Professionnel – David

Bienvenue sur mon portfolio de développeur Front-End.

Ce projet présente mon parcours, mes compétences, mes réalisations ainsi que plusieurs moyens de me contacter.

## 📸 Aperçu

Le portfolio est composé de plusieurs sections :

- Hero
- À propos
- Compétences
- Mon Parcours
- Mes Projets
- Contact
- Footer

Le site est entièrement responsive et compatible avec les modes clair et sombre.

---

# 🚀 Technologies utilisées

- HTML5
- CSS3
- JavaScript Vanilla
- Tailwind CSS
- EmailJS
- Font Awesome

---

# ✨ Fonctionnalités

## Interface

- Responsive Design
- Dark / Light Mode
- Animations fluides
- Glassmorphism
- Lazy Loading des images

## Contact

Le formulaire permet :

- Validation en temps réel
- Vérification des champs
- Sélection d'un indicatif international
- Choix du motif de contact
- Envoi via EmailJS
- Contact direct via WhatsApp
- Spinner de chargement
- Messages de succès / erreur
